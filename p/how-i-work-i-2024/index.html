How i work i 2024

Getting stuff done

Apr 14, 2024
1 minute read

Almost 2 years to the day i update this post with what is my current setup.

Hardware

I have replaced the Lenovo Thinkpad T430s with a Lenovo ThinkBook 14 which i chose based on what it weights. It is what i use most of the time for work.

I’ve also replaced the Trackball with the most current version of the Logitech MX Ergo, mostly because it is bluetooth enabled. I’ve also purchased on a whim a Ducky ONE 2 Mini which has rekindled my taste for smaller keyboards and is currently my main keyboard.

I also got into flight sims and driving sims. The flying is done using a VKB Gladiator NXT and a Thrustmaster HOTAS Warthog Dual Throttle. Driving is accomplished using a Thrustmaster T300 Ferrari Edition with the TH8A shifter. I’ve also added a basic usb 3.0 hub from Amazon to connect all of this.

I also acquired an used Oxygen 25 MK IV for which i’m still trying to find the time to explore.

Software

I’ve taken an interest in Cybersecurity for which i assigned my old laptop to run Kali Linux exclusively. I live inside Notion, Todoist and VS Code.
